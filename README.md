# purr-image

PurrBot ImageAPI Javascript Wrapper

## Installation

```bash
npm i purr-image
```

## Usage

CommonJS:

```js
const  ImageAPI  =  require("purr-image");
```

ES6:

```ts
import  ImageAPI  from  "purr-image";
```

## Build

```bash
git clone https://github.com/justapieop/purr-image
cd purr-image
npm run build
```
